Django Social Media Site

A simple Social Site

A simple social media network clone-site project built with Django

It has key features such as;
a) Groups(similar to facebook),
b) Multiple Users and Authorizations (Register if new, user Login),
c) Add posts in groups(similar to a tweet),
d) Linking User profiles with '@'_symbol,
e) Multiple user applications such as;
View the feeds,
Share the posts, and
Comment on posts.

Admin of the site has control over:
Designing and developing the website.
Creating and updating the database,
Inserting, viewing, updating, deleting the data in the database.

Technologies used for this python project are:
1. Front-end:
HTML, CSS, Bootstrap4, and Javascript.
2. Back-end:
Python, Mysql, and Django.
3. Software requirements:
Text editor especially Pycharm and Python3-IDLE.

More documented notes on how the program works, are found in 'notes.txt' file attached on this repository in github
